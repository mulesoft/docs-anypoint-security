= Add a Certificate
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

Add a single PEM certificate to the secret group. This certificate is the "leaf" certificate (end-entity certificate). 

. In the *Secret Groups* list view, select the secret group, and click *Edit*. 
. In the menu on the left, select *Certificate*, and then click *Add Certificate*. 
. In the *Add Certificate* screen, add the required information. 
+
* *Name* +
Enter a name for your certificate.
* *Type* +
Select the certificate type from the drop-down menu.
+
Supported types are:
+
** PEM
* *Certificate File* +
Click *Choose File*, and select the certificate file to upload.
* *Override Expiration Date* +
(Optional) Select the date to override the default expiration date of the certificate.
. Click *Save*.
