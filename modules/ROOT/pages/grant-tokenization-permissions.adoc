An administrator for your organization can grant management permissions to any user by using access management.

== Tokenization Permissions

A user must have a set of predefined permissions to create and manage tokenization formats and services:

[%header%autowidth.spread,cols="a,a"]
|===
| Name | Description
| Manage Tokenization Formats
| Provides the ability to create, modify, and delete tokenization formats to use with the tokenization service in a specific environment.

| Manage Tokenization Services
| Provides the ability to create and deploy a tokenization service, as well as modify and delete a tokenization service in a specific environment.
|===

== Grant a User Tokenization Permissions

. In the Access Management page, go to the left navigation menu and click *Users*.
. To grant permissions to a user, click the user's name. 
. In the *Permissions* tab, click *Tokenization*. 
. Select *Manage Tokenization Formats* from the *Permissions* menu, and click the blue "plus" icon image:blue-plus-security.png[]. 
. Select the environment from the Environment menu, then select the *Manage Tokenization Services* permission from the Permissions menu, and click the blue "plus" icon image:blue-plus-security.png[].
