= Add a TLS Context for Anypoint Runtime Fabric (Advanced Configuration)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

If your environment requires advanced configuration such as client authentication (mutual authentication), selecting ciphers, and selecting TLS versions, you can import a TLS context from the secrets manager. Refer to xref:1.13@runtime-fabric::configure-adv-tls-context.adoc[Import a TLS Context from Secrets Manager (Advanced)].
